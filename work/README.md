# work/

Aquí vive lo que estás construyendo ahora mismo.

## Qué va aquí

- PRDs en draft
- Documentación de tu producto que está viva
- Proyectos en curso
- Decisiones que estás tomando, con su sustento
- Análisis hechos a la medida de un problema actual

## Regla mental

**Si lo estás editando esta semana, va aquí.**

## Qué NO va aquí

- Material crudo recibido → `input/`
- Cosas listas para publicar afuera → `output/`
- Ideas sueltas sin proyecto activo → `thinking/`

## Cómo organizar adentro

Subcarpeta por proyecto:

```
work/
├── proyecto-roadmap-q2/
│   ├── prd-feature-x.md
│   ├── analisis-competencia.md
│   └── decisiones.md
├── proyecto-onboarding/
│   └── ...
```

Si tu proyecto crece, el agente puede ayudarte a reorganizar. Pídeselo cuando una carpeta tenga más de 10-15 archivos.

## Cuándo mover a `output/`

Cuando algo está listo para que otra persona lo lea como producto terminado — post publicable, reporte para ejecutivos, propuesta para cliente. El agente NO mueve archivos solo, tú decides.
