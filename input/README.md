# input/

Aquí vive lo que llega de afuera y no creaste para este brain.

## Qué va aquí

- Notas de reunión, transcripciones (de Granola, Otter, Zoom, etc.)
- Research que descargaste, papers, links guardados
- Documentación externa que necesitas como contexto
- Inspiración, screenshots, snippets sueltos

## Regla mental

**Si lo recibiste, va aquí. Punto.**

## Qué NO va aquí

- Tu trabajo activo → `work/`
- Cosas que vas a publicar → `output/`
- Ideas sueltas tuyas sin destino → `thinking/`

## Limpieza

Cuando algo de aquí ya fue procesado (resumido, archivado, incorporado a un proyecto), bórralo o muévelo. `input/` no es archivo histórico — es bandeja de entrada.

## Ejemplo incluido

`_ejemplo-transcripcion.md` — una transcripción de reunión inventada que sirve para probar el skill `process-transcript`. Bórrala cuando ya no la necesites.
